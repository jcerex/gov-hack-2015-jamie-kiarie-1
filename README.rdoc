== README

This project is being developed for Gov Hack 2015 in Canberra.
Contributors: Jamie Cerexhe & Kiarie Ndegwa

We are using the 'Counts of Businesses including Entries and Exits, 2010-2014  MetaData : Businesses by Industry Division' database from ABS.Stat to help people identify whether the industry they work in is shrinking and then suggests related industries which have been growing and offers re-skilling opportunities.
